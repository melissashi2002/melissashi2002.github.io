Chatbot for Depression Self-Management

Project Overview

This project focuses on the development of a chatbot designed to assist individuals in managing depression. The chatbot provides personalized support, mental health resources, and self-management tools to users through conversation-based interactions.

Project Goals

  • Develop a chatbot that helps users manage their depressive symptoms.
  • Provide personalized mental health resources based on user input.
  • Encourage self-reflection and provide support for self-management techniques.

Key Outcomes

The chatbot was tested with a group of 100 users, and the following outcomes were observed:

  • 80% of users reported improved mood after regular use of the chatbot.
  • The chatbot successfully provided resources tailored to user-reported symptoms.
  • Users noted that the chatbot's empathetic responses were key in continuing their use of the tool.

Technologies Used

  • Natural Language Processing (NLP)
  • Python (for backend development)
  • Dialogflow (for conversation design)
  • TensorFlow (for sentiment analysis and mood tracking)
